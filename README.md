# App Shell V2

A ready-to-use design scaffold for building HighLevel products.

---

## Getting started

1. Clone this repo and open it in Cursor
2. Open the AI chat — **Cmd+L** (Mac) or **Ctrl+L** (Windows)
3. Make sure you're on **claude-3.5-sonnet**
4. Type exactly: **set up my project** — then hit Enter

The AI will ask you a few questions and set everything up for you.

---

After setup, all your work goes in `src/pages/`. That's the only folder you'll ever need to touch.
